Welcome to Mindmap

!

This app has no autosave. Closing or refreshing the tab without saving will permanently destroy your work. Save early and often using the button in the top-right corner.

Nodes

Click + Node to add a card. Drag to reposition. Double-click to open its editor — set title, body text, colour, and border style. Attach files from the editor too.

Pins

Click + Pin to place a point-down marker. Drag to move. Double-click to open settings. Useful as waypoints or silent connection anchors.

Lines

Hover near an element's edge to reveal its dock, then drag to another element to draw a line. Pin-to-pin lines are straight; all others curve. Double-click a line to edit it.

Attachments

Inside the node or pin editor, drag files and images into the drop zones. Double-click any attachment thumbnail to view it fullscreen.

Mouse Controls

  • Click — select (orange ring)
  • Ctrl+Click — add to selection
  • Click+Drag on empty space — draw selection box
  • Double-Click — open editor
  • Right-Click+Drag — pan canvas
  • Scroll — zoom in / out

Keyboard Shortcuts

  • Del — delete selected
  • Ctrl+C — copy selected
  • Ctrl+V — paste
  • Esc — clear selection

Save & Load

Use Save (top-right) to download the project as a zip. The name in the centre-top box becomes the filename. Use Load to reopen a saved zip — the filename box will update automatically.
