Welcome

An interactive geographic mind-map. Pin places, write nodes, draw connections — all anchored to real lat/lng coordinates on a live tile map.

Quick Start

  • Right-click + drag to pan the map. Scroll to zoom.
  • Toolbar + Pin / + Node creates an element at the current center.
  • Single-click any element to open its editor in the right sidebar.
  • Double-click a node to flip it (front title ↔ back metadata).
  • Hover near an element (not directly on it) — cursor turns to crosshair — then click-drag to another element to draw a connecting line.
  • Click a line to reveal its endpoint handles — drag handles to re-anchor along edges or to another element.
  • Drag files into the editor's drop zone to attach images and documents.

Storage

Your work lives in this browser's local memory. Browser cleanup utilities or clearing site data will wipe it. Back up often using the Save button.
